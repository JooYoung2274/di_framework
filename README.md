## 목적

Express로 Nest.js와 같이 런타임 때 알아서 의존성을 찾아서 주입해주는 DI 프레임워크 만드는게 목적임.

## 기록

-   20230503 -> 일단 기초적인 IoC 컨테이너 관련 로직 작성 완료. 일단 app.ts에 몰아서 테스트 진행했으니 이제 폴더구조 변경해야 할듯

## 실행 방법

```bash
# 의존성 설치
$ npm install
# 실행
$ npm run dev
```
